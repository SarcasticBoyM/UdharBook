# Shop Admin Guide

## Dashboard

The dashboard shows:

- Total customers
- Total pending amount
- Today follow-ups
- Overdue follow-ups
- Recovery stats
- Staff activity

## Customers

Shop Admins can:

- Add customers
- Search and filter customers
- Record payments
- Add notes
- View customer history
- Send WhatsApp reminders
- Export reports

## Follow-ups

Use Follow-ups to:

- Schedule exact date and time
- Set priority: Low, Medium, High, Urgent
- Track status: Pending, Completed, Missed, Rescheduled
- Add call notes, reminder notes, and customer response
- View today, weekly, upcoming, and overdue work

## Excel Import

Use Upload Excel to import customers for the active shop. Duplicate contact numbers are checked inside the shop only.

## Notifications

Allow browser notifications when prompted. UdharBook will notify before scheduled follow-ups and for missed follow-ups.

## Security

Shop Admins and Staff only see data from their assigned shop. Super Admins can switch shops from the sidebar.
